
# 🏎️ Zenix Service Suite v1.15.1 [BETA / HARDWARE EDITION]
(SUPPORT END)
**Zenix Service Suite v1.15.1** is my very first, historic beta version of a PC optimizer, completely written in classic **Windows Batch scripting (.bat)**! 

This tool is designed for basic OS cleanup, resolving network latency, and gathering system specifications directly through the Windows command line.

---

## ⚡ Main Features of v1.15.1 Beta:
* 🛠 **MODULE_GAME** — Force closes heavy background Riot Games launchers and performs a deep cleanup of NVIDIA Shader Cache (`DXCache`) to stabilize FPS.
* 🌐 **MODULE_NET** — Resets the network stack (`winsock reset`, `int ip reset`) and flushes the ARP table to eliminate lag spikes and high ping.
* ⚙️ **MODULE_ZENIX** — Safely removes junk cache files from system folders and Discord.
* 📊 **PC Hardware Passport** — Fast native poll of motherboard, CPU, and GPU models using Windows console utilities.

---

## ⚠️ Developer's Note (Archive Lore)
This is an **old-school Batch version** of the project from May 14-16. Due to the limitations of the legacy CMD engine, it might experience text encoding issues or temporary internet drops during a hard network reset.

**Zenix Service Suite v1.15.1** — это моя самая первая, историческая бета-версия геймерского оптимизатора, полностью написанная на классических скриптах **Windows Batch (.bat)**! 

Софт создан для базовой зачистки скрытого мусора операционной системы, исправления сетевых задержек и быстрого сбора характеристик твоего компьютера через командную строку Windows.

## ⚡ Главные фишки ветки v1.15.1:
* 🛠 **MODULE_GAME** — Автоматическая выгрузка тяжелых фоновых лаунчеров Riot Games и зачистка кэша шейдеров NVIDIA (`DXCache`) для стабилизации FPS.
* 🌐 **MODULE_NET** — Системный сброс сетевого стека (`winsock reset`, `int ip reset`) и ARP-таблицы для тотального уничтожения лагов и скачков пинга.
* ⚙️ **MODULE_ZENIX** — Безопасная очистка кэша системных папок и мессенджера Discord.
* 📊 **Характеристики ПК** — Быстрый нативный опрос материнской платы, процессора и видеокарты напрямую через консольные службы Windows.
